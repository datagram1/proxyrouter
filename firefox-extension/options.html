ProxyRouter Settings

Proxy Server Configuration

Hostname or IP address of your ProxyRouter server
HTTP proxy port (default: 8080)

API Configuration

Hostname for ProxyRouter API (usually same as proxy host)
API port (default: 8081)

Routing Configuration

LOCAL: Bypass proxy for direct connections
GENERAL: Use randomly selected healthy proxy from pool
TOR: Route through Tor network (requires Tor daemon)
UPSTREAM: Use specific proxy from database
Automatically choose routing mode based on destination (experimental)
